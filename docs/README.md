## 📖 代码注释

项目中的所有核心代码都包含了详细的中文注释：

### Go 后端代码

所有核心模块都有详细的注释说明：

- `main.go` - 服务启动流程和模块初始化
- `internal/config/` - 配置管理和环境变量处理
- `internal/rag/` - RAG 核心业务逻辑
- `internal/http/` - HTTP 请求处理
- `internal/embedding/` - 文本向量化
- `internal/vectorstore/` - 向量数据库操作
- `internal/llm/` - 大语言模型调用
- `internal/limit/` - 限流器实现
- `internal/model/` - 数据模型定义

每个文件都包含：
- 模块级注释（package 说明）
- 类型定义注释
- 函数/方法注释（参数、返回值、工作原理）
- 关键逻辑的行内注释

### Python 构建脚本

- `build_db.py` - 向量数据库构建流程
- `embedding.py` - BGE-M3 模型封装
